Guides

Guide: How to hide the notch on android 9 pie. 10th of May 2019

First go to settings tap system then tap on about phone, and tap the build number 7 times. This will enable developer options. Now go back to system then head over to developer options. Find drawing tap on display cutout then tap hide. Now the notch on your device should be hidden.

Guide: How to run a game made in unity on the Asus Tinkerboard, Raspberry PI and other ARM boards. 11th of May 2019

To do this you'll need to Compile your game for WebGL Then copy the folder to your device. Now open a terminal window in the game folder. We will now start a web server using the one built in to Python now in the terminal window copy and paste this if you have python 2 installed. python -m SimpleHTTPServer 8000 or if you have python 3 installed then use this instead. python3 -m http.server Then open chromium and type this in the address bar localhost:8000 and now you can play your game!

Tips

Tip: Check what fonts websites are using in Firefox and Chrome. 11th of June 2019

Right Click on your Mouse or Trackpad then click Inspect Element. For Example if you Click on the main elment in the Inspector on this website you'll see that the H tags and P tags are using Noto sans then the default Sans serif font on your device.

Tip: Fix Blurry sprites in Unity. 21th of June 2019

First off in unity open the folder where you store your sprites. Then click on your sprite. In the inspector find Filter Mode and change it from bilinear to pont (no filter). When you unselect the sprite from the inspector you'll get a popup asking you to apply changes. Click apply and now the sprite will no longer look blurry.
